tmux

cheat sheet · Ctrl+a config · zsh aliases
Prefix: Ctrl+a  —  press this before any tmux command key  ·  Alt+arrow navigates panes without a prefix
Sessions — in & out
tn <name>New named session
ta <name>Attach to session
tlsList all sessions
tk <name>Kill session by name
tmux-hereNew session named after current dir
Ctrl+a dDetach (leave running)
Ctrl+a $Rename current session
Ctrl+a :kill-sessionKill from inside
Tip: Detach is almost always what you want — session keeps running.
Windows (tabs)
Ctrl+a cNew window
Ctrl+a nNext window
Ctrl+a pPrevious window
Ctrl+a 0–9Jump to window by number
Ctrl+a ,Rename current window
Ctrl+a &Kill current window
Ctrl+a wVisual window picker
Windows start at 1 in this config (not 0).
Panes (splits)
Ctrl+a |Split vertical
Ctrl+a -Split horizontal
Alt+←↑↓→Navigate panes (no prefix!)
Ctrl+a zZoom pane (toggle fullscreen)
Ctrl+a xKill current pane
Ctrl+a qShow pane numbers
Ctrl+a { / }Swap pane left/right
Tip: Ctrl+a z to zoom into logs, then z again to restore.
Dev session layout — project workflow
dev From project dir — opens 3-pane layout and attaches (reattaches if exists)
dev <name> <path> Named session from any directory
Pane 1 (top-left) Claude Code — claude auto-started
Pane 2 (bottom-left) Shell in project dir
Pane 3 (right) App / logs — npm run dev or equivalent
Running dev when the session exists just reattaches — nothing is clobbered.

Misc
Ctrl+a ?Show all keybindings
Ctrl+a tShow clock
Ctrl+a :Command prompt
Ctrl+a rReload config
exit / Ctrl+dClose current pane/shell
Closing things — when to use what
Ctrl+a d Detach — leave session running, exit terminal safely. Best default.
exit in pane Close that pane's shell. Last pane closes the window; last window ends the session.
tk <name> Kill session from outside — destroys layout and all processes in it.
Ctrl+a :kill-session Kill session from inside.
